Componente Didactica

Aprende con el proyecto de como esta organizado el mercado, como disponibilizar los datos en formato opendata o desarrollo de aplicaciones y mucho más.

Priceproforinfosys

Plataforma que unifica y centraliza la información para realizar previsionesy predicciones para el Mercado de Electricidad. En ella se recoje toda la información de los siguientes sistemas.

Accesso a la plataforma

La información es obtenida de..

omieinfosys

Información disponibilizada por el Operador de Mercado. Procesamos la información en energía producida por tecnologías y precio del mercado diario, que además fija los valores del precio para los futuros negociados en otros mercados financieros. Tiene un carácter financiero aunque contiene, en su proceso de casacion de precios, restrinciones "a priori" del Operador de Sistema.

Descarga datos Ver más

reeinfosys

Información disponibilizada por el Operador de Sistema. Tiene un caracter fisico-Ingenieril. La información se disponibiliza de forma agregada, ya que para el operador de sistema es necesario saber todas las unidades que están produciendo energía independientemente del mercado donde esa unidad la ha negociado. Esta página, además, disponibiliza alguna información que podemos considerar de input para el mercado: * Las previsiones de Demanda Horaria * y de la generación eólica.

Descarga datos Ver más

Unidadesinfosys

La Información disponibilizada para las unidades de producción no esta centralizada. Aquí pretendemos gestionar dos tipos de información distinta: Unidades de producción y las ofertas que realizan. Hay que rellenar la información con varias fuentes.

ATWS

Se ha estudiado el Método Académico de HWTES Holt-Winters Triple Exponential Smoothing. Se ha aplicado este modelado a los precios del mercado (con una vision de serie Temporal) para obtener un método predictivo. Este modelo está en backtesting.

Modelo de los Vecinos

Este problema solo esta en formulación pero la idea para este método es construir una métrica que nos permita comparar días similares y construir previsiones de mercado usando combinaciones lineales de dichos días.

Modelo de Lógica difusa

Este problema solo esta en formulación pero la idea para este método es construir reglas difusas coherentes con el funcionamiento del mercado para definir funciones de base para el modelado y cálculo de previsiones del Mercado Diario.

Algoritmo Genético y optimización

Este problema solo está en formulación pero la idea esta relacionada con el modelo de Logica difusa. La evolucion de cada función base de precio para las tecnologías en el Mercado Diario, son distintas para hora. Con esta analise se pretende establecer una métrica que permita entender esta evolución en el Mercado al largo del historico.
